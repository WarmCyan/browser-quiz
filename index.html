Usage

This site takes one or more lists of things/movements to practice, randomizes them, and generates a timed quiz.

Lists

To create or edit lists, go to the "Edit" tab. The yellow "+" button will add a new list. Each list consists of a single line name field, and a text area where you can enter the movements associated with that list. Each line in the text area is a separate item/will be brought up as an invidiual quiz question.

When you have created a few lists and you want to save the whole set/re-use them later, hit the "save" button in the upper right. This will download your lists as a JSON file. To reload them, click the "load" button, and select your JSON file.

If there are items in your list that might take a variable/long time to run, you can put a "*" at the end of the line. When these items come up in the quiz, rather than automatically moving on after some number of seconds, a button will appear for you to manually advance the quiz to the next item.

Quizzes

When you have some lists and you're ready to BUCKLE DOWN AND GET STUDYING, head over to the "Quiz" tab. For an individual quiz session, you can enable/disable which lists to include by checking/unchecking them.

There are two sliders/timer controls for the quiz session, "execution time" and "reset time". Execution time is the amount of time you have after the next item is shown to complete it. In between each item there's a brief grace period to "reset", controlled by the "reset time".

When you're ready to go, and have the timers set to your liking, hit the "Quiz me!" button!

Happy practicing!
